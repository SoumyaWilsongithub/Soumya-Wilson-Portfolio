NotesApplication

The Challenge: Standard local storage solutions (like SQLite) do not allow users to access their data across different devices or sessions.

The Implementation: I developed a robust backend using Spring Boot with @RestController to manage full CRUD operations (Create, Read, Update, Delete). I used Spring Data JPA and MySQL to ensure all user notes are securely stored and retrievable via a cloud-sync experience.

Architecture: The project follows professional software architecture patterns, featuring structured REST endpoints for seamless integration between the Android frontend and the backend database.
