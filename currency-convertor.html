Currency-Converter

The Challenge: Dealing with asynchronous data from third-party APIs can lead to UI "freezing" or inaccurate calculations during fluctuating market rates.

The Implementation: I used Asynchronous JavaScript (ES6+) and the Fetch API to parse live currency datasets into a dynamic, comparative table. The application features modular logic to handle over 100+ global currencies, updating the DOM instantly without requiring a page reload.

The Design: I designed a professional multi-page interface using HTML5 and CSS3, including dedicated 'About' and 'Contact' modules, focusing on clean navigation and a mobile-first user experience.
