SimpleCalc - Android App

The Project: SimpleCalc is a clean and straightforward calculator app built for Android. The goal of this project was to create a reliable tool that performs everyday math—like addition and multiplication—with a focus on speed and accuracy.

How I Built It: I used Java and Android Studio to build the "engine" that processes the numbers. I spent a lot of time making sure that the app is easy to use; the buttons are large and clear, and the math results show up instantly as you type.

The Challenge: The trickiest part was "error handling." I wrote code to make sure the app doesn't close or show an error if a user accidentally types two plus signs in a row or tries to divide by zero. It’s all about making the app feel smooth and "unbreakable" for the user.
