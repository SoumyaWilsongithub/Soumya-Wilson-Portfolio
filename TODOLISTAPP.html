The Project: I created a digital notebook where you can save, edit, and delete your thoughts or daily tasks. Unlike very basic apps that lose data when you close them, this app is designed to store your information permanently in a professional database.

How I Built It: I used Spring Boot and MySQL to build the storage system. I wrote the backend logic so that every time you add a note, it is sent to the MySQL database. I used REST APIs to make sure the app can quickly "read" and "write" your data whenever you need it.

Why It Matters: This project taught me how to handle CRUD (Create, Read, Update, Delete) operations, which are the building blocks of almost every professional app. It shows that I can build a reliable system that manages data safely and keeps things organized for the user.
