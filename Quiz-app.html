Quiz App

The Challenge:The main difficulty in a quiz app is managing the "State"—tracking scores accurately, preventing users from changing answers once submitted, and handling the countdown timer without crashing the app during orientation changes.

The Implementation: I developed this application using Java and Android Studio, implementing a robust logic controller to randomize questions and validate user selections in real-time. I used View Binding for efficient UI interaction and implemented a custom Result Activity that calculates performance percentages and provides instant feedback.

UX Design:I focused on a "Gamified" experience by adding progress bars to show quiz completion status and integrated a countdown timer to increase user engagement. The app uses a clean, modern XML layout with custom button states to indicate correct and incorrect answers immediately.
