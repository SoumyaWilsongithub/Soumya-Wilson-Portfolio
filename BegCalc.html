Calculator Application

The Challenge: The primary challenge in building a functional calculator is handling user input sequences—preventing multiple decimal points in a single number, managing "division by zero" errors, and ensuring the UI remains responsive during continuous calculations.

The Implementation: Developed using Java and Android Studio, I implemented a custom parsing logic to handle arithmetic operations (Addition, Subtraction, Multiplication, Division). I utilized ConstraintLayout to ensure the calculator interface adapts seamlessly to various screen sizes and orientations.

Architecture: The application follows a clean, modular architecture with separate components for UI logic, input handling, and calculation processing. This design ensures scalability and maintainability.
